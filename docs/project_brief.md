# Smartan Command Centre

## Project Overview

Smartan Command Centre is a command-line management platform designed to serve as the operational backbone of Smartan House.

The system centralises member management, Acorn Project tracking, competency assessments, and organisational reporting into a single platform.

The goal is to replace fragmented records stored across spreadsheets, forms, notebooks, and messaging platforms with a structured and scalable system.

---

## Problem Statement

Smartan House requires a reliable method for:

* Managing Smartan records
* Tracking Acorn Projects
* Recording competency assessments
* Generating organisational reports
* Preserving institutional knowledge

Without a centralised system, data becomes difficult to maintain, analyse, and scale.

---

## Objectives

The project aims to demonstrate:

* Object-oriented software design
* Data modelling
* File-based persistence
* Input validation
* Exception handling
* Testing
* Software architecture principles

---

## Version 1 Scope

The first version includes:

### Member Management

* Create members
* View members
* Update members
* Delete members
* Search members

### Acorn Project Management

* Create projects
* View projects
* Update projects
* Track project status

### Competency Assessments

* Record assessments
* View assessment history
* Track competency development

### Reporting

* Member summaries
* Project summaries
* Top performers
* Organisational metrics

---

## Long-Term Vision

Future versions may evolve into a national-scale youth operating system capable of managing:

* Multiple communities
* Fellowship programmes
* Conferences
* Mentor networks
* Incubation programmes
* Large-scale impact reporting

The current version focuses on building strong architectural foundations for future expansion.
